# Personal Microsite

A static one-page microsite served via GitHub Pages.
Not intended for public distribution — search-engine indexing is disabled.

## Files

| File | Purpose |
|------|---------|
| `index.html` | Main page (all images embedded as base64) |
| `splash.mp4` | Loop intro video |
| `transition.mp4` | One-shot intro transition |
| `robots.txt` | Crawler exclusion |
| `.nojekyll` | Skip Jekyll processing |

## Deploy

Static site — push to GitHub and enable Pages
(`Settings → Pages → Source: main / root`).
